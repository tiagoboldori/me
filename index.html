Projetos



Controle de estoque e agendamentos (ERP)

Sistema capaz de realizar agendamento de aulas e controle de mantimentos, desenvolvido para suprir necessidades do campus
IFRJ nilo peçanha. Projeto desenvolvido usando principalmente PHP, MySQL, css, html e JavaScript.

Servidor de Chat em tempo real

O projeto tinha como objetivo final uma aplicação que permitisse troca de mensagens ao vivo e foi desenvolvido usando C/C++. Ao final do desenvolvimento o projeto ja contava com diversas funcionalidades, entre elas: suporte a conexão de múltiplos usuários simultâneamente (através de multithreading), desenvolvimento de um protocolo personalizado a nível de aplicação e indentificação e diferenciação dos usuários conectados. Desenvolvido exclusivamente para plataformas Windows (WinSock).

Snake Game

Jogo da cobrinha simples desenvolvido usando a biblioteca PyGame. Projeto desenvolvido no começo da minha jornada na graduação. Foi interessante para treinar estruturas de dados mais simples como a lista encadeada e também importante para treinar a lógica de programação. O projeto usou exclusivamente Python.

Planejamento Universitario

Projeto simples para gerenciar turmas de um ambiente acadêmico imaginário. Desenvolvido usando principalmente C# e SQL Server - além da stack básica para desenvolvimento web.
